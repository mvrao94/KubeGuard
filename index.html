Secure Your Kubernetes Clusters, Effortlessly.

KubeGuard is a lightweight, developer-first security scanner that finds common misconfigurations in your Kubernetes manifests and live clusters.

Get Started on GitHub

Why KubeGuard?

Shift security left and empower your developers with a tool designed for modern CI/CD workflows.

Static Manifest Scanning

Catch issues before they are deployed. KubeGuard scans your YAML files for privileged containers, missing resource limits, and more.

Live Cluster Scanning

Continuously monitor your running applications. KubeGuard uses the Kubernetes API to inspect live resources for active vulnerabilities.

CI/CD Integration

Automate security gates in your pipelines. KubeGuard is a lightweight container that easily integrates with Jenkins, GitLab, and more.

Simple REST API

Built with Java and Spring Boot, KubeGuard exposes a clean, intuitive REST API for triggering scans and retrieving results.

A Two-Pronged Approach to Security

KubeGuard provides defense-in-depth by scanning at every stage of the development lifecycle.

1. Static Scan (Pre-Deployment)

In your CI/CD pipeline, KubeGuard parses your Kubernetes YAML files directly. It acts as a quality gate, failing the build if it finds critical issues before insecure code ever reaches your cluster.

$ kubeguard --scan ./manifests --fail-on-critical

2. Live Scan (Post-Deployment)

Once deployed, KubeGuard runs as a microservice inside your cluster. It uses the Kubernetes API to continuously monitor running workloads, catching manual misconfigurations or drift.

$ curl http://kubeguard/api/scan/cluster/prod
