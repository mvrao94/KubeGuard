Enterprise-Grade Kubernetes Security, Made Simple.

KubeGuard is a comprehensive security scanner that implements CIS Kubernetes Benchmark recommendations to protect your clusters through static analysis and runtime monitoring.

Get Started on GitHub

Why KubeGuard?

Shift security left and empower your developers with a tool designed for modern CI/CD workflows.

Advanced Static Analysis

Catch issues early with 25+ security rules implementing CIS Kubernetes Benchmark recommendations. Identifies privileged containers, exposed secrets, and insecure configurations.

Runtime Protection

Continuously monitor your clusters with live scanning, RBAC analysis, and network policy validation. Get real-time alerts on security drift and misconfigurations.

Enterprise Integration

Production-ready with built-in Prometheus metrics, health checks, and PostgreSQL persistence. Seamlessly integrates with CI/CD pipelines and existing security tools.

Developer Experience

RESTful API with OpenAPI 3.0 spec and Swagger UI. Async processing, detailed reporting, and actionable remediation advice for each finding.

Production-Ready Security

KubeGuard delivers enterprise-grade security with a modern, cloud-native architecture designed for reliability and scale.

1. Automated Security Gates

Integrate with your CI/CD pipeline using our REST API. Block deployments with critical security issues and track findings over time with detailed JSON reports and metrics.

$ curl -X POST http://kubeguard/api/v1/scan/manifests -d '{"path": "./k8s"}'

2. Runtime Defense

Deploy KubeGuard as a highly available service with autoscaling, network policies, and RBAC. Monitor live clusters with our comprehensive rules engine and get instant alerts.

$ curl http://kubeguard/api/v1/scan/cluster/default
